# إضافة حساب كأدمن

إذا ظهرت رسالة **"هذا الحساب غير مسجل كأدمن"** فالحساب الذي سجّلت الدخول به غير مضاف في جدول الأدمن.

## الخطوات

### 1. فتح Supabase
- ادخل إلى [Supabase Dashboard](https://supabase.com/dashboard) ومشروعك.

### 2. الحصول على البريد أو الـ user_id
- من القائمة: **Authentication** → **Users**.
- انسخ **البريد الإلكتروني** للحساب الذي تريد جعله أدمن، أو انسخ **User UID**.

### 3. تشغيل SQL
- من القائمة: **SQL Editor** → **New query**.
- الصق الأمر التالي وغيّر البريد إلى بريدك:

```sql
INSERT INTO public.admin_users (user_id)
SELECT id FROM auth.users WHERE email = 'بريدك@example.com'
ON CONFLICT (user_id) DO NOTHING;
```

- اضغط **Run**.

### 4. إعادة المحاولة
- ارجع إلى صفحة لوحة التحكم وسجّل الخروج ثم الدخول مرة أخرى، أو حدّث الصفحة.

بعد تنفيذ الـ SQL بنجاح، نفس الحساب سيُعتبر أدمن وستختفي رسالة "غير مسجل كأدمن".
