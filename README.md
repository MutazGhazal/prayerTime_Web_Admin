# Prayer Time App

This repository contains the initial scaffolding for a mobile app that shows
daily prayer times and can share a daily status to WhatsApp.

## Structure
- `docs/` Project documentation and requirements.
- `mobile/` Flutter mobile app (MVP scaffolding).

## Next Steps (MVP)
- Load prayer times from Aladhan API.
- Display times for today with next prayer countdown.
- Local caching for offline use.
- Schedule notifications for each prayer.
- Generate a daily status image and share to WhatsApp (Android Intent).
