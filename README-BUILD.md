# بناء لوحة الأدمن للإنتاج

## التطوير المحلي
```bash
npm install
npm run dev
```

## بناء الإنتاج
```bash
npm run build
```
سيتم إنشاء مجلد `dist/` يحتوي على الملفات الجاهزة للنشر.

## النشر على GitHub Pages
1. نفّذ `npm run build`
2. انشر محتويات مجلد `dist/` على الفرع `gh-pages` أو المجلد المناسب
3. تأكد أن ملف `public/config.js` يحتوي إعداداتك الصحيحة (يُنقل كما هو إلى dist)

## ملاحظات
- **config.js**: عدّل ملف `public/config.js` بالإعدادات الخاصة بك قبل البناء
- **مسار النشر**: إذا كان الموقع على مسار فرعي (مثل `/prayerTime_Web_Admin/`)، تأكد أن `base` في `vite.config.js` مطابق للمسار
